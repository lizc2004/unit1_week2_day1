# unit1_week2_day1
# Primo esercizio in JavaScript

Corso: Epicode - Unit 1, Week 2, Day 1

Contenuti dell'esercizio:
- Dichiarazione variabili con `let`
- Tipi di dato primitivi (string, number, boolean, null, undefined)
- Operatori di confronto `===` e `!==`
- Riassegnazione di variabili
- Uso di `console.log()` per verificare le risposte

## Come eseguire

1. Apri `index.html` nel browser
2. Premi F12 per aprire la console
3. Ricarica la pagina e osserva i valori stampati
